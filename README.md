#  Power BI Sales Analysis Dashboard

##  Overview

This project presents a **Power BI dashboard** built to analyze sales and customer data. It provides clear insights into revenue, customer behavior, and product performance to support data-driven decision-making.

---

##  Dataset

The dataset includes key business fields such as:

* Transaction details (ID, Date)
* Customer information (ID, Name, Segment)
* Product details (ID, Category, Price)
* Sales metrics (Quantity, Total Amount, Profit)
* Region & Payment Method

---

##  Key Features

* Interactive dashboard with filters and slicers
* KPI tracking (Revenue, Profit, Customer Spend)
* Category & product-wise performance analysis
* Customer segmentation insights
* Time-based trends (monthly/quarterly analysis)

---

##  Insights

* Identifies **top-performing products and categories**
* Highlights **high-value customers**
* Tracks **sales trends over time**
* Shows **regional performance differences**
* Helps detect **profitability patterns**

---

##  Tools Used

* Power BI (Data Modeling & Visualization)
* DAX (Calculated Measures & KPIs)
* Excel / Data Source Integration

---

##  How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Refresh data (if needed)
4. Explore dashboard using filters

---

##  Conclusion

This dashboard enables quick and effective analysis of sales data, helping businesses improve performance, optimize strategies, and identify growth opportunities.
